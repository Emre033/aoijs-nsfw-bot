# Discord NSFW bot

## 🎊 Nasıl Kullanılır
1. `main.js` dosyasında gereken alanları doldurun.
2. `npm i` ve `node .` komutlarıyla botu başlatın.
3. Botu sunucunuza ekleyin.

## 🧨Destek
Eğer problem yaşıyorsan [Discord Sunucumuza](https://discord.gg/akparti) katılabilirsin.
